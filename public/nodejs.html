Node.js alapok

A Node.js egy JavaScript runtime, ami lehetővé teszi a JavaScript futtatását a szerveroldalon. A Node.js segítségével könnyedén tudunk webszervereket, API-kat, vagy akár parancssori alkalmazásokat készíteni.

A Node.js a V8 JavaScript engine-re épül, ami a Google Chrome böngészőben is megtalálható. Ebben a bemutatóban lépésenként készítünk egy egyszerű projektet.

Előszügségletek

Az alábbi dolgokra lesz szükségünk a projekt elkészítéséhez:

  • Node.js telepítve a számítógépünkre
  • Egy szövegszerkesztő, például Visual Studio Code
  • Alapvető JavaScript ismeretek

Lépések

Mappa előkészítése

Nyissunk egy az adott opercáiós rendszernek megfelelő terminált, és hozzunk létre egy új mappát a projektnek:

mkdir nodejs101 cd nodejs101

npm inicializálása

Adjuk ki az alábbi parancsot a mappa inicializálásához:

npm init

Itt meg kell adnunk néhány alapvető információt a projektünkről, például a nevét, a verziót, a leírást, stb. Ezeket később is módosíthatjuk a package.json fájlban.

package.json beállítása

Az inicializálás után a package.json fájlban találjuk a projektünk beállításait. Itt tudjuk megadni a függőségeket, scripteket, és egyéb információkat a projektről.

Alapvetően a package.json beállítása a következőképpen néz ki:

{ "name": "nodejs101", "version": "1.0.0", "description": "Egy egyszerű Node.js projekt", "main": "index.js", "scripts": { "test": "echo \"Error: no test specified\" && exit 1" }, "keywords": [], "author": "", "license": "ISC" }

Az alapértelmezett beállításokon kívül itt találhatóak a scriptek, amiket a npm run scriptNév paranccsal tudunk futtatni. Ezeket a scripteket a scripts objektumban tudjuk megadni.

Mivel szeretnénk, hogy futtatható legyen a projektünk, így scripts objektumba hozzáadunk egy start scriptet is:

{ "name": "nodejs101", "version": "1.0.0", "description": "Egy egyszerű Node.js projekt", "main": "index.js", "scripts": { "test": "echo \"Error: no test specified\" && exit 1", "start": "node index.js" }, "keywords": [], "author": "", "license": "ISC" }

Projekt futtatása

A projekt futtatásához adjuk ki a következő parancsot a terminálban:

npm start
